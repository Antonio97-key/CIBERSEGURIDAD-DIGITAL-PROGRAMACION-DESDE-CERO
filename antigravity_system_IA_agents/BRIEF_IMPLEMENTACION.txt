SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════════

PROBLEMA 1: "ModuleNotFoundError: No module named 'antigravity_system'"
  SOLUCIÓN:
    1. Verifica estar en carpeta antigravity_project
    2. Verifica que venv está activado: (venv) debe estar en prompt
    3. Reinstala: pip install -r requirements.txt

PROBLEMA 2: "Python version 3.10 or earlier"
  SOLUCIÓN:
    1. Desinstala Python actual
    2. Descarga Python 3.11+ desde python.org
    3. Instala e intenta nuevamente

PROBLEMA 3: "pip: command not found"
  SOLUCIÓN:
    1. Reinstala Python incluyendo pip
    2. O usa: python -m pip install ...

PROBLEMA 4: "Permission denied" en macOS/Linux
  SOLUCIÓN:
    chmod +x antigravity_system/main.py
    python antigravity_system/main.py

PROBLEMA 5: "Database connection error"
  SOLUCIÓN:
    Cambiar en .env:
    DATABASE_URL=sqlite:///antigravity.db
    (No necesita base de datos externa para desarrollo)

PROBLEMA 6: "Port already in use"
  SOLUCIÓN:
    Cambiar puerto en config/system_config.py si es necesario

PROBLEMA 7: Los imports fallan
  SOLUCIÓN:
    1. Verificar que __init__.py existen en todas las carpetas
    2. python -c "import antigravity_system" (prueba)
    3. Reinstalar todo desde cero si es necesario
