# Development Workflow

Step 1 — Product Definition
Product Manager defines the feature.

Step 2 — Architecture
Architect Agent defines system architecture.

Step 3 — Backend Development
Backend Agent implements APIs and logic.

Step 4 — Frontend Development
Frontend Agent builds UI.

Step 5 — UI/UX Optimization
UI/UX Agent improves visual experience.

Step 6 — Security Review
Security Agent audits the system.

Step 7 — Quality Assurance
QA Agent tests and verifies stability.
