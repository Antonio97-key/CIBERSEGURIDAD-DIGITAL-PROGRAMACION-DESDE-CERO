# Global AI Rules

These rules apply to ALL agents.

General Principles:
- Follow clean architecture
- Maintain modular code
- Avoid duplicated logic
- Document important decisions

Technology Guidelines:
Frontend:
- React / Next.js
- TailwindCSS
- TypeScript

Backend:
- Node.js
- Express or Fastify
- PostgreSQL

Security:
- Use authentication standards
- Protect against common web attacks

Coordination:
- Architect Agent validates architecture decisions
- Product Agent defines features
- QA Agent validates final code quality
